# Ring Jump Game

タイミングを合わせてジャンプするシンプルなアクションゲームです。

## 遊び方 / How to Play

1. **スタート**: スペースキーを押すか、画面をクリックしてゲームを開始します。
2. **待機**: ランダムな時間（1〜4秒）待つと、合図の音が鳴り始めます。
3. **ジャンプ**: 音が鳴り始めてから **3.3秒後** にリングが落下します。
    - **3.0秒 〜 3.3秒** の間にジャンプしてください。
    - 3.0秒より早いと「早すぎ」で失格です。
    - 3.3秒を過ぎるとリングと一緒に落ちてゲームオーバーです。

## 技術スタック / Technologies

- HTML5
- CSS3 (3D Transforms)
- JavaScript (Web Audio API)

## インストール / Installation

このリポジトリをクローンし、`index.html` をブラウザで開くだけでプレイできます。

git clone https://github.com/Naoya79/ring_jump_game.git
